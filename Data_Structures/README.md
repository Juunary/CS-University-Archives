# Data Structures

A collection of data structure application projects implemented in C++. Each project implements core data structures to solve real-world problems.

## Project List

### [01_Music_Playlist_BST](./01_Music_Playlist_BST/)
BST-based Music Playlist Management System

- Composed of Title BST, Artist BST, Playback Queue, and Playlist (Linked List)
- Features for adding/searching/deleting songs and creating playlists

---

### [02_Employee_DB_BPlusTree](./02_Employee_DB_BPlusTree/)
B+ Tree-based Employee Database

- Employee information indexing with B+ Tree (name search, range search)
- Sorting and department-wise aggregation using Selection Tree (Tournament Tree) and Max Heap

---

### [03_Graph_Algorithm_Engine](./03_Graph_Algorithm_Engine/)
Graph Algorithm Engine

- Supports two graph representations: Adjacency Matrix and Adjacency List
- Implementation of BFS, DFS, Dijkstra, Kruskal, Bellman-Ford, Floyd-Warshall, and Centrality

---

## Languages and Environment

- Language: C++
- Build: G++ (C++11 or later)
