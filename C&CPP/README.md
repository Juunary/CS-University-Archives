# C & C++ Programming Labs

A collection of weekly programming labs covering C language fundamentals through data structures.

## Weekly Breakdown

| Week | Date | Topic | Main Practice Content |
|------|------|-------|----------------------|
| Week 01 | 09/08 | Basics | Swap using pointers, array sorting, sieve of Eratosthenes |
| Week 02 | 09/15 | Strings & Arrays | Custom string functions, 2D arrays, hotel reservation system |
| Week 03 | 09/22 | Dynamic Memory | Dynamic allocation with malloc/realloc, blackjack game |
| Week 04 | 09/29 | Strings | Case conversion, string interleaving, Roman numeral conversion |
| Week 05 | 10/13 | File I/O | File input/output, word parsing and sorting |
| Week 06 | 10/22 | Files & Apps | GPA calculation, menu ordering/receipts, word list sorting |
| Week 07 | 10/27 | Structs & Algorithms | Student score management, date calculations, King Stone movement |
| Week 08 | 11/03 | Structs & Enums | Union, Enum, library checkout system |
| Week 09 | 11/10 | Projects | Admin login, cafe ordering system, library inventory |
| Week 10 | 11/17 | Linked Lists | Singly linked list basic operations, odd/even separation |
| Week 11 | 11/24 | Advanced Linked Lists | Doubly linked lists, multilevel linked list sorting |

## Language & Environment

- Language: C
- Compiler: GCC
